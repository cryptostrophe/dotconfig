# my

## Synopsis

```sh
my [ip|localip|router|ssid]
```

## Description

`my` is a tool for examining network connections.

## Usage

```sh
> my
Public IP: 203.0.113.42
 Local IP: 10.0.1.6
Router IP: 10.0.1.1
     SSID: Pythia
> my ip
203.0.113.42
> my localip
10.0.1.6
> my router
10.0.1.1
> my ssid
Pythia
```
