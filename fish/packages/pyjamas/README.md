# PYJamas

Move comfortably between **P**roperty lists, **Y**AML, and **J**SON.

Supports filename arguments and/or standard input:

| Command      | Files | STDIN |
| ------------ | :---: | :---: |
| `json2plist` |   ✓   |   ✓   |
| `json2yaml`  |   ✓   |   ✓   |
| `plist2json` |   ✓   |       |
| `plist2yaml` |   ✓   |       |
| `yaml2json`  |   ✓   |   ✓   |
| `yaml2plist` |   ✓   |   ✓   |

**Note:** `plist2*` requires macOS's `plutil(1)`.
